讨论列表

get /project/discuss/list

请求参数：

参数名称 说明 类型 是否必须
id 项目id int 是

返回参数:

参数名称 说明 类型 是否必须
id ID int 是
problem 问题 String 是
commenter 评论人 String 是
comment_time 评论时间 int 是
replyer 回复人 String 是
reply_time 回复时间 int 是
status 是否公开 int 是

说明：status:0/不公开，1/公开
